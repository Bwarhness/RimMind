## ğŸš€ Quick Start

### Installation

1. **Subscribe on Steam Workshop**  
   [RimMind on Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3666997391)

2. **Enable the mod**  
   - In RimWorld, go to **Mods**
   - Find **RimMind** in your mod list
   - Check the box to enable it
   - Restart RimWorld

3. **Get an OpenRouter API Key**  
   - Visit [OpenRouter](https://openrouter.ai/)
   - Create a free account
   - Generate an API key from your dashboard

4. **Configure RimMind**  
   - In RimWorld, go to **Options â†’ Mod Settings â†’ RimMind**
   - Paste your OpenRouter API key
   - Choose your preferred AI model (optional)
   - Save settings

### Basic Usage

1. **Open the AI Chat**  
   Press the **RimMind button** in your colony interface (or use the hotkey if configured)

2. **Ask Questions**  
   ```
   "What's the mood of my colonists?"
   "Do I have enough food for winter?"
   "What should I research next?"
   "Which colonists are best suited for combat?"
   "Show me my current resource stockpiles"
   ```

3. **Get Strategic Advice**  
   ```
   "Help me plan defenses for a raid"
   "What are my colony's biggest risks right now?"
   "Should I accept this quest?"
   "Which colonist should I prioritize for medical treatment?"
   ```

The AI will analyze your colony's actual data and provide informed, context-aware responses.
